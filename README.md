# 📽 Netflix Clone
  Almost complete Frontend clone of the Netflix platform with the use of a local API and zustand.

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
## 👨🏾‍💻 Technologies
* React
* Next.js
* Bootstrap
* Zustand
